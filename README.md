# Silver Mode Site

A polished, scroll-based concept website for **Silver Mode** — a senior-friendly assisted-driving interface concept focused on confidence, dignity, and independence.

## Project structure

- `index.html` — page structure and all copy/content sections.
- `styles.css` — visual design system, responsive layout, and animation styling.
- `script.js` — scroll-triggered interactions and interactive mockup behavior.

## How to preview locally

Because this is a static site, you can run it in either of these beginner-friendly ways:

1. **Quick open:** double-click `index.html` to open in your browser.
2. **Local server (recommended):** from this folder run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## How to edit

1. Update section text and structure in `index.html`.
2. Tweak colors, spacing, and responsive behavior in `styles.css`.
3. Adjust scroll interactions and mockup logic in `script.js`.
